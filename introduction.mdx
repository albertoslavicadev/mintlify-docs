## What is Plumifly SDK?

Plumifly SDK provides React components to display your Plumifly blog content. It handles all the data fetching and presentation, allowing you to focus on customization and integration.

<Note>
  While the SDK provides ready-to-use React components, we also offer a RESTful API for those who:
  - Are not using React or Next.js
  - Want to build custom components
  - Need more control over data fetching and rendering
  Check out our [API Reference](/api-reference/introduction) for direct API integration.
</Note>

## Example Usage

Display your blog content in seconds:

```tsx
import { BlogList, initPlumifly } from 'plumifly-sdk';

// Initialize the SDK with your API key
initPlumifly({
  apiKey: 'your-api-key'
});

// Display your blog posts
export default function BlogPage({ data }) {
  return (
    <BlogList 
      data={data}
      theme={{
        colors: {
          primary: '#007AFF'
        }
      }}
    />
  );
}
```
